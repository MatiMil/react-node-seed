# How to build 

1. install `webpack` globally. run `npm i -g webpack`
2. change to your project directory
3. run `npm i`
4. run `build`

# How to develop 

1. install `webpack` globally. run `npm i -g webpack`
2. change to your project directory
3. run `npm i`
4. run `npm run dev`
5. open your browser on `http://localhost:8080`

# How to run server 

1. run `npm i`
2. run `npm run start`
5. server will run on `http://localhost:3000`
